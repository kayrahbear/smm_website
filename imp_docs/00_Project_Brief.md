# Ashley Brewer Books — Project Brief

## Project
Create a launch website for Ashley Brewer's debut children's book, *The Great Ghost Mix Up*, in the *Savannah Manor Mysteries* series.

## Initial Goal
Publish a polished coming-soon page to `ashleybrewerbooks.com` while the full site is designed and built.

## Tech Direction
- Next.js
- Tailwind CSS
- Static export
- AWS hosting

## Why this stack
- Fast and lightweight
- Easy to iterate on as the series grows
- Strong SEO foundation
- Supports interactive UI without requiring a full app backend
- Can be hosted cleanly on AWS

## Phase 1 Deliverables
- Project skeleton
- Coming soon homepage
- Initial visual direction
- Hosting plan
- Obsidian-friendly planning docs
