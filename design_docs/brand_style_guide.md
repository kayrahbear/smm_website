## Typography

Ashley requested a font similar to the speech text in the book.

### Primary Display Font (Headings)

Suggested Options:

- **Baloo 2**
- **Fredoka**
- **Luckiest Guy**
- **Chewy**

These feel playful and children's-book friendly.

### Body Font

For readability:

- **Nunito**
- **Quicksand**
- **Poppins**
